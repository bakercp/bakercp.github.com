40 class Period
41  // A Period represents an abstract amount of time. In some cases a concrete
42  // number of microseconds can be calculated (e.g. for DateTimeFields such
43  // as MICROSECOND, MILLISECOND, SECOND, MINUTE, HOUR), but for some measures
44  // of time (e.g. HOUR, DAY, WEEK, MONTH, YEAR), it is only possible to
45  // calculate the equivalent number of microseconds relative to a known
46  // calendar start time. This is because concepts such as "MONTH" and "YEAR"
47  // are vary due to "leap" seconds, and other calendrical irregularities.
48  //
49  // Empty or NULL periods are those that have all fields set to zero.
