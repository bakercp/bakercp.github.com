Detailed Description

This structure is used to share data between the Libav library and the client video application. This shall be zero-allocated and available as AVCodecContext.hwaccel_context. All user members can be set once during initialization or through each AVCodecContext.get_buffer() function call. In any case, they must be valid prior to calling decoding functions.
